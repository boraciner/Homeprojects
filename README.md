### Install it from scratch

1. Clone the repository
	```    
	$ git clone https://github.com/boraciner/Homeprojects.git
	```
2. Enter into the folder
  	```    
	$ cd Homeprojects
	```
3. Run Docker Compose
  	```    
	$ docker compose up
	```
